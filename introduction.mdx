**Mintlify Components** (`@mintlify/components`) is an open-source React UI library that gives you 22 production-ready components specifically designed for documentation sites. Built on top of Tailwind CSS, every component ships with sensible defaults, full dark mode support, and accessibility baked in — so you spend your time writing great docs, not rebuilding UI primitives from scratch.

## What's included

The library exports the following components and utilities:

| Component | Description |
|---|---|
| `Accordion` / `Accordion.Group` | Collapsible content sections |
| `Badge` | Inline status and label badges |
| `Callout` (Info / Warning / Note / Tip / Check / Danger) | Highlighted callout blocks |
| `Card` | Content cards with optional icons and links |
| `CodeBlock` | Syntax-highlighted code with copy support |
| `CodeGroup` / `CodeGroupSelect` | Tabbed multi-language code snippets |
| `Color` / `Color.Row` / `Color.Item` | Color swatch palettes |
| `Columns` | Multi-column layout helper |
| `Expandable` | Toggleable content regions |
| `Frame` | Bordered image / media frames |
| `Icon` | Icon rendering utility |
| `Mermaid` | Mermaid.js diagram renderer |
| `Panel` | Visually separated content panels |
| `Property` | API property definition rows |
| `Search` | Search input component |
| `Steps` / `Steps.Item` | Numbered step-by-step guides |
| `Tabs` / `Tabs.Item` | Tabbed content sections |
| `Tile` | Large navigational tiles |
| `Tooltip` | Hover tooltips |
| `Tree` / `Tree.File` / `Tree.Folder` | File-tree visualizations |
| `Update` | Changelog / release-note entries |
| `cn` | Class-name utility (wraps `clsx` + `tailwind-merge`) |

## Requirements

<Note>
  Make sure your environment meets all of the following requirements before
  installing.

  | Requirement | Version |
  |---|---|
  | Node.js | >= 20 |
  | React | 18 or 19 |
  | Tailwind CSS | v3 or v4 |
</Note>

React and Tailwind CSS are **peer dependencies** — you need to have them installed in your project before adding `@mintlify/components`.

Ready to get started? Head over to the [Installation guide](/installation) to add the library to your project.
