## Get started in three steps

Follow these steps to install the package, wire up your styles, and render your first component.

<Steps>
  <Step title="Install the package">
    Add `@mintlify/components` to your project using your preferred package
    manager.

    ```bash
    npm install @mintlify/components
    ```
  </Step>
  <Step title="Set up Tailwind CSS styles">
    Import the component stylesheet **before** your Tailwind CSS import so that
    design tokens are available to every component.

    ```css
    @import "@mintlify/components/styles.css";
    @import "tailwindcss";

    @theme {
      --color-primary: #0ea5e9;
    }
    ```

    <Note>
      The `@mintlify/components/styles.css` import must always appear before any
      Tailwind directive. Reversing the order will cause styles to be overridden
      incorrectly.
    </Note>
  </Step>
  <Step title="Import and use a component">
    Import any component directly from the package and drop it into your JSX.

    ```tsx
    import { Callout, Accordion } from '@mintlify/components';

    export default function Page() {
      return (
        <Callout type="info">
          You just rendered your first Mintlify component!
        </Callout>
      );
    }
    ```

    <Tip>
      All components are tree-shakeable — only the components you import will be
      included in your production bundle.
    </Tip>
  </Step>
</Steps>
