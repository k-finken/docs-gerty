Versions are listed newest-first so you can immediately see what changed in the most recent release. Each entry notes the publish date and categorizes changes as **Added**, **Changed**, or **Fixed**.

## 1.0.0 — January 23, 2025

### Initialized components library

The first stable release of `@mintlify/components`. This version ships the complete foundational set of components:

| Component | Description |
|---|---|
| `Accordion` | Expandable content sections |
| `Badge` | Status and label badges |
| `Callout` | Highlighted information boxes |
| `Card` | Content cards |
| `CodeBlock` | Syntax-highlighted code with copy button |
| `CodeGroup` | Code block container with tabs or dropdown selection |
| `Columns` | Multi-column layouts |
| `Expandable` | Collapsible sections |
| `Frame` | Image and content frames |
| `Icon` | Icon rendering |
| `Mermaid` | Diagram rendering |
| `Panel` | Panel layouts |
| `Property` | API property display |
| `Steps` | Step-by-step instructions |
| `Tabs` | Tabbed interfaces |
| `Tile` | Tile grids |
| `Tooltip` | Hover tooltips |
| `Tree` | Hierarchical tree views |
| `Update` | Changelog entries |
